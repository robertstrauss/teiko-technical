## Starting the App

Prepare and start the Python database backend:
```sh
$ pip install -r requirements.txt
$ chmod +x load_data.sh # in case permissions fail to copy on some systems
$ ./load_data.sh [port] # port optional, default is 8000
```

Prepare and run the React app frontend:
```sh
$ npm install && npm run build
$ npm start
```
